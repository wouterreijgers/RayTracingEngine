# RayTracingEngine
A Raytracing engine capable of generating realisctic images. This raytracer was build for the course Computer Graphics. It contains all the basic aspects of a raytracing engine and some extras:
- Add solid textures
- Image textures
- Export .jpg files
- Expert .mp4 videos
